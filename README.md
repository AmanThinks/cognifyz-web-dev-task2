# 🚀 Responsive Card Layout  
### Cognifyz Web Development Internship – Level 2 Task 2

This project is developed as part of the **Web Development Internship at Cognifyz Technologies**.

The goal of this task was to build a **responsive card layout using HTML, CSS, and Media Queries** that adapts smoothly across different devices such as desktops, tablets, and mobile screens.

The project also focuses on creating a **modern UI with interactive elements and smooth hover animations**.

---

# 📌 Project Overview

The website showcases different **technology domains** using interactive cards.  
Each card includes:

✔ Image  
✔ Title  
✔ Description  
✔ Interactive button  
✔ External link for more information  

When users click a card button, they are redirected to a webpage where they can learn more about that specific technology.

---

# ✨ Features

✅ Fully Responsive Layout

✅ Modern Dark Theme UI
